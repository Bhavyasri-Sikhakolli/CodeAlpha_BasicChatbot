# Basic Chatbot

## Description

A simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs and continues the conversation until the user types "bye".

## Features

* Responds to greetings
* Answers simple questions
* Provides predefined replies
* Ends the conversation when the user types "bye"

## Technologies Used

* Python

## Concepts Used

* Functions
* Loops
* Conditional Statements (if-elif-else)
* User Input and Output

## How to Run

1. Open the project folder in VS Code.

2. Open the terminal.

3. Run the following command:

   py chatbot.py

4. Interact with the chatbot through the terminal.

## Sample Inputs

* hello
* how are you
* what is your name
* good morning
* thank you
* bye
